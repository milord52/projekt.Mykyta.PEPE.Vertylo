# projekt.Mykyta.PEPE.Vertylo

Projekt edukacyjny wykonany w języku Python z wykorzystaniem frameworka Flask.

## Opis
Aplikacja webowa analizująca dane dotyczące stylu życia użytkownika.
Projekt porównuje dwa różne modele decyzyjne:
- model regułowy,
- model punktowy.

## Technologie
- Python 3
- Flask
- HTML
- CSS

## Uruchomienie
1. Zainstaluj wymagane biblioteki:
   pip install flask
2. Uruchom aplikację:
   python app.py
3. Otwórz w przeglądarce:
   http://127.0.0.1:5000
